Discover Our Powerful API Endpoints

Seamlessly integrate advanced functionalities into your applications with our robust and well-documented API services.

GET /api/bug/forcecall

🚀 Force Call Initiation - This endpoint allows you to trigger a simulated force call action to a specified target. Ideal for testing and emergency communication systems or for internal system alerts.

Parameters: target=string (required, phone number with country code, e.g., `6281234567890`).
Authentication: None (for demonstration)
Response: JSON indicating success or failure.

GET /api/bug/carousels

🎠 Carousel Message Delivery - Send interactive carousel messages to a specified target, perfect for rich content delivery, marketing campaigns, or showcasing product listings.

Parameters: target=string (required, phone number), fjids=string (required, comma-separated unique identifiers for carousel items).
Example: /api/bug/carousels?target=6281234567890&fjids=item1,item2,item3
Authentication: None (for demonstration)

POST /api/send-message

✉️ Custom Message Dispatch - Send customizable text messages to any valid recipient. Supports various message formats and media attachments.

Request Body (JSON): { "target": "string", "message": "string", "mediaUrl": "string (optional)" }
Authentication: API Key (Bearer Token)

GET /api/status/system

📊 System Status Check - Retrieve real-time operational status and health metrics of the API platform, including uptime, resource utilization, and active connections.

Response: JSON object with various system metrics (e.g., CPU usage, memory, uptime, API latency).
Authentication: None

GET /api/user/profile/:id

👤 User Profile Retrieval - Fetch public profile information for a specific user by their unique ID.

Parameters: :id=string (required, User ID).
Authentication: API Key (Bearer Token)

POST /api/broadcast

📢 Broadcast Message to Multiple Targets - Send a single message to a list of specified recipients simultaneously.

Request Body (JSON): { "targets": "array", "message": "string" }
Authentication: API Key (Bearer Token)
