Diverse API Endpoints

Explore a selection of our powerful RESTful APIs designed for seamless integration.

GET /api/forcecall

🚀 Force Call Initiation - This endpoint allows you to trigger a simulated force call action to a specified target. Ideal for testing and emergency communication systems.

Parameters: target (required, phone number with country code).
Example: /api/bug/forcecall?target=6281234567890

GET /api/carousels

🎠 Carousel Message Delivery - Send interactive carousel messages to a specified target, perfect for rich content delivery and marketing campaigns.

Parameters: target (required, phone number), fjids (required, unique identifiers for carousel items).
Example: /api/bug/carousels?target=6281234567890&fjids=item1,item2,item3

POST /api/send-message

✉️ Custom Message Dispatch - Send customizable text messages to any valid recipient. Supports various message formats.

Body: { "target": "string", "message": "string" }
Authorization: Bearer Token (API Key)

GET /api/status/system

📊 System Status Check - Retrieve real-time operational status and health metrics of the API platform.

Returns: JSON object with system uptime, memory usage, etc.
